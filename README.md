# Build Lerna Apps

This library helps to build all React Apps if lerna changed.

## Installation

Use the "go get".

```bash
go get github.com/malikov0216/buildApp
```

## Usage

```go
import (
   "fmt"
   "github.com/malikov0216/buildApp"
)

func main () {
   buildApp()
}
```
